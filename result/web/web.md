# 웹 성능 예산 및 개선 결과

## 웹 성능 예산

```
# 정량 기준
- 텍스트 압축을 통해 자바스크립트 파일의 크기가 1MB를 넘지 않아야 한다.
- 사용하지 않는 자바스크립트 파일은 지연로딩 설정한다.

# 시간 기준
- FCP 1.8초 미만
- SI 2.3초 미만
- LCP 2.1초 미만
- TTI 2.2초 미만

# 규칙 기준
- pagespeed 기준으로 성능 지표가 80점을 넘겨야 한다.
- 서울교통공사의 성능 지표인 82점을 넘겨야 한다.
```
